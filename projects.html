TREX GAME

The Trex Game is a game where we create one trex jumping over obstacles and trying to dodge them. I got the idea of the trex game from when my interent went off and the game used to pop up in my web browser! It took 8 classes to make this game with a lot of effort but in the end I learnt a lot and it was a lot of fun! I also converted my trex game into a mobile game via an app known as Thunkable.

The Car Racing Game is a game where I created a car racing game thriugh learning database and firebase. The concepts that I learnt: I made them alive by coding that game. Its basically a game of 4 car racers who compete other players to reach the finish line of the game. If you want to play the game with your friends, open the tab 4 times

Geologist is of helping a geologist finishing a project about objects having properties like restitution, friction and stiffness. This was one of my favourite projects and it was very fun to see the output! All the objects were having real life properties!

JUMPING BOX

Oh yes! Jumping Box, a sprite project where a box is jumping over platforms. When box lands on the platform the colour of the box becomes the colour of the platform! And the box stops at a random platform.
